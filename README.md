 Анализ текста

Простая программа на Python для анализа текста.

 Возможности

* Подсчёт общего количества слов
* Поиск самого частого слова
* Подсчёт уникальных слов
* Очистка текста от знаков препинания
* Приведение текста к нижнему регистру

 Технологии

* Python 3

 Как запустить

1. Убедись, что установлен Python
2. Запусти файл:

   ```bash
   python main.py
   ```
3. Введи текст в консоль

 Пример работы

Ввод:

```
мир мир привет мир
```

Вывод:

```
Самое частое слово: мир
```

Структура проекта
```
main.py       # основной файл программы
README.md     # описание проекта
```

 Описание

Программа принимает текст от пользователя, обрабатывает его и выводит основную статистику.
Подходит для обучения основам работы со строками и словарями в Python.
